**DonTeploEnergy** — это учебный pet-проект, разработанный в рамках производственной практики. Цель проекта — создать прототип веб-платформы для учета, оповещение, информированния пользователя. ## Стек технологий

Проект построен на современном технологическом стеке, который охватывает все этапы разработки — от кода до деплоя:

**Бэкенд:**
- **Go** — высокопроизводительный язык для написания серверной логики и API.
- **PostgreSQL** — реляционные базы данных для хранения информации о пользователях и тарифах.

**Фронтенд:**
- **React** — библиотека для построения динамичного пользовательского интерфейса.
- **TypeScript** — для типобезопасности и повышения качества кода.
- **CSS** — для адаптивной верстки и стилизации компонентов.

**Инфраструктура и автоматизация:**
- **Docker & Docker Compose** — полная контейнеризация всех сервисов для упрощения развертывания.
- **Nginx** — веб-сервер для проксирования запросов к бэкенду и раздачи статических файлов фронтенда.

<a id="func"></a>
## Функциональные возможности

Проект находится в стадии разработки, но уже реализован следующий базовый функционал:

- [ ] **Авторизация и ролевая модель:** Разделение прав для администраторов, операторов и обычных пользователей.
- [ ] **Заявление:** Добавление, редактирование и просмотр заявление от пользователей.
- [ ] **Тарифы:** Добавление, редактирование и просмотр тарифов.
- [ ] **Филиалы:** Просмотр действующих филлиалов предприятия.
- [ ] **Вопрос-ответ** FAQ для пользователей.
- [ ] **Личный кабинет:** Просмотр чата переписки с диспечеом, изменение.
- [ ] **Административная панель (FrontAdmin):** Управление пользователями, тарифами и справочной информацией.
